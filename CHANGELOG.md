# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project aims to

## [Unreleased]

### Added

- Initial open-source release: Slack-authenticated form that logs submissions to
  Google Sheets and posts to Slack, plus the Slack Photo Flair tool.
- Documentation site under `/docs` (overview, getting started, configuration,
  how it works, deploy).
- AI agent tooling: `AGENTS.md`, `CLAUDE.md`, `.claude/` commands, agent, and
  skills, and a `prompts/` library.
- Community health files: `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`,
  issue/PR templates, Dependabot, and a build CI workflow.
